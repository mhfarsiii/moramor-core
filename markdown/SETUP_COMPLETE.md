```bash

# بعد از دریافت کد ۶ رقمی:
curl -X POST http://localhost:3000/api/v1/auth/verify-code \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@moramor.com","code":"123456"}'
```

## 📋 API Endpoints اصلی

### احراز هویت (Auth)
- `POST /api/v1/auth/send-code` - ارسال کد تأیید به ایمیل
- `POST /api/v1/auth/verify-code` - تأیید کد و دریافت توکن‌ها
- `POST /api/v1/auth/refresh` - تمدید توکن
- `GET /api/v1/auth/me` - اطلاعات کاربر جاری

### محصولات (Products)
- `GET /api/v1/products` - لیست محصولات
- `POST /api/v1/products` - ایجاد محصول (نیاز به ادمین)
- `GET /api/v1/products/:id` - جزئیات محصول
- `PATCH /api/v1/products/:id` - ویرایش محصول
- `DELETE /api/v1/products/:id` - حذف محصول

### دسته‌بندی (Categories)
- `GET /api/v1/categories` - لیست دسته‌بندی‌ها
- `POST /api/v1/categories` - ایجاد دسته‌بندی (نیاز به ادمین)

### سبد خرید (Cart)
- `GET /api/v1/cart` - مشاهده سبد
- `POST /api/v1/cart` - افزودن به سبد
- `DELETE /api/v1/cart/clear` - خالی کردن سبد

### سفارشات (Orders)
- `GET /api/v1/orders` - لیست سفارشات
- `GET /api/v1/orders/:id` - جزئیات سفارش
- `POST /api/v1/checkout` - ثبت سفارش

### آدرس‌ها (Addresses)
- `GET /api/v1/addresses` - لیست آدرس‌ها
- `POST /api/v1/addresses` - ایجاد آدرس

### علاقه‌مندی‌ها (Wishlist)
- `GET /api/v1/wishlist` - لیست علاقه‌مندی‌ها
- `POST /api/v1/wishlist` - افزودن به علاقه‌مندی‌ها

### سلامت سیستم (Health)
- `GET /api/v1/health` - بررسی وضعیت سرور

## 🛠️ دستورات مفید

### توقف سرور:
```bash
lsof -ti:3000 | xargs kill -9
```

### شروع مجدد سرور:
```bash
npm run start:dev
```

### مشاهده لاگ‌ها:
```bash
tail -f /tmp/moramor-server.log
```

### توقف Docker Containers:
```bash
docker-compose -f docker-compose.dev.yml down
```

### شروع Docker Containers:
```bash
docker-compose -f docker-compose.dev.yml up -d
```

### اعمال Migration جدید:
```bash

```

### Seed دیتابیس:
```bash
npm run prisma:seed
```

## 📝 راهنمای تست سریع

1. **باز کردن محیط تست:**
   - فایل `api-tester.html` در مرورگر باز شده است
   
2. **ورود به سیستم:**
   - از منوی سمت راست "احراز هویت > ورود" را انتخاب کنید
   - روی "ورود با ادمین" کلیک کنید
   - دکمه ورود را بزنید
   - توکن به صورت خودکار ذخیره می‌شود

3. **مشاهده محصولات:**
   - "محصولات > لیست محصولات" را انتخاب کنید
   - دکمه "دریافت لیست" را بزنید

4. **ایجاد محصول جدید:**
   - "محصولات > ایجاد محصول" را انتخاب کنید
   - یک Category ID از لیست دسته‌بندی‌ها کپی کنید
   - اطلاعات محصول را وارد کنید
   - دکمه "ایجاد محصول" را بزنید

5. **افزودن به سبد:**
   - یک Product ID از لیست محصولات کپی کنید
   - "سبد خرید > افزودن به سبد" را انتخاب کنید
   - Product ID را وارد کنید
   - دکمه "افزودن به سبد" را بزنید

6. **ایجاد آدرس:**
   - "آدرس‌ها > ایجاد آدرس" را انتخاب کنید
   - اطلاعات آدرس را وارد کنید
   - دکمه "ایجاد آدرس" را بزنید

7. **ثبت سفارش:**
   - Address ID را از لیست آدرس‌ها کپی کنید
   - "پرداخت > پرداخت سفارش" را انتخاب کنید
   - Address ID را وارد کنید
   - روش پرداخت را انتخاب کنید
   - دکمه "ثبت سفارش" را بزنید

## 🐛 عیب‌یابی

### اگر سرور راه‌اندازی نمی‌شود:
1. پورت 3000 را آزاد کنید: `lsof -ti:3000 | xargs kill -9`
2. Docker را راه‌اندازی کنید: `open -a Docker`
3. سرور را مجدد راه‌اندازی کنید: `npm run start:dev`

### اگر دیتابیس خطا می‌دهد:
1. Docker containers را مجدد راه‌اندازی کنید
2. Migration ها را دوباره اجرا کنید: `npm run prisma:migrate`
